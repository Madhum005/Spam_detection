# 📩 SMS Spam Classification

A Machine Learning project that classifies SMS messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques and Logistic Regression.

## 🚀 Features

* Text preprocessing and cleaning
* Feature extraction using NLP
* Spam/Ham classification
* Model evaluation and performance analysis

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn
* Jupyter Notebook

## 📂 Project Structure

```text
SMS-Spam-Classification/
│
├── Copy_of_SPAM_CLASSIFICATION__DEC_2025.ipynb
├── README.md
└── requirements.txt
```

## ▶️ Run the Project

```bash
pip install -r requirements.txt
jupyter notebook
```

## 📊 Output

The model predicts whether an SMS message is:

* ✅ Ham (Legitimate Message)
* 🚫 Spam (Unwanted Message)
